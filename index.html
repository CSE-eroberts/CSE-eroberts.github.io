CSCE 242: Web Applications

CSCE 242 focuses on the fundamentals of web development using HTML and CSS. The course introduces how websites are structured and how content is displayed in a browser. Students learn best practices for organizing files and writing clean, readable code.

https://github.com/CSE-eroberts/CSE-eroberts.github.io

Assignments

Assignment 1 - Basic HTML

An overview of color psychology, including lists, tables, and supporting images that explain how color influences mood.

Assignment 2 - Basic CSS

A styled hiking trail guide, photo row, and trail table that highlights updated access in South Carolina.

Assignment 3 - Page Layout

A layout that practices spacing, typography, and clean structure using reusable layout styles.

Assignment 4 - USC Page

A university homepage recreation emphasizing layout, typography, and component styling.

Assignment 5 - JavaScript Interactions

An interactive three-column layout with geometry, date selection, and image swapping features.

Assignment 6 - Class Countdown

A responsive two-exercise page with a toggle menu, slider messages, and a real-time countdown to class at 8:30 AM.

Assignment 7 - Fish Tank

A CSS art fish tank with animated fish and JavaScript-generated bubbles created with a for loop and random positions.

Assignment 8 - Arrays

An interactive mood selector that displays song links from associative arrays and embeds a YouTube video after selecting a song.
